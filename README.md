# epam-parking-designing
Design and implement a parking lot management system in Pytthon. The system will manage two types of parking.

# Problem Description

Design and implement a parking lot management system in Python. The system will manage two types of spots (Compact, Regular) and two types of vehicles (Motorbike, Car).

1-) Parking Rules:

* A Motorbike can park in either Compact or Regular spots

* A Car can only park in Regular spots

* When parking, place vehicles in the first available valid spot with the lowest ID
